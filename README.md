# 🚗 Car O'phile

> **Discover the pure numbers defining ultimate speed.**

Car O'phile is a visually immersive, single-page website that showcases the **10 fastest production hypercars in the world**. Designed with a futuristic cyberpunk aesthetic, the website combines detailed vehicle information with smooth animations, interactive effects, and responsive design to provide an engaging browsing experience for automobile enthusiasts.

---

## 📌 Project Overview

The website presents a curated collection of the world's fastest hypercars along with:

- Vehicle descriptions
- Top speed statistics
- Performance highlights
- Interactive speed visualizations
- Comparison table
- Live search functionality
- Modern UI animations

The project is built entirely using **HTML**, **CSS**, and **Vanilla JavaScript** without any external frameworks.

# ⚠ Known Issues

Some minor issues that could be improved:

- Search input initially uses a non-standard `type="placeholder"` and is converted to `text` via JavaScript.
- A few footer links contain malformed URLs (for example, the Rimac link).
- Some HTML structure (such as placing `<h1>` inside a `<p>`) is not semantically valid.
- Navigation links are placeholders and do not point to functional pages.
- And many more.....
